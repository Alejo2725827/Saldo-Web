# Termos e Condições de Uso - Saldo

Última atualização: 13 de junho de 2026.

## 1. Introdução
Ao usar o Saldo, você aceita estes termos. Se não concordar, por favor, não utilize o aplicativo.

## 2. Diferenciação de Planos
- **Plano Gratuito (Free)**: A sincronização de dados é manual. É responsabilidade do usuário realizar a sincronização (Uplink) para fazer backup de suas alterações na nuvem.
- **Plano Premium**: Inclui sincronização automática em tempo real.

## 3. Responsabilidade do Usuário
É de responsabilidade exclusiva do usuário:
- Manter suas credenciais de acesso seguras.
- Realizar sincronizações manuais periódicas se for usuário do Plano Gratuito para evitar perda de dados em caso de falha ou perda do dispositivo.
- Gerenciar seus lembretes locais, os quais não são sincronizados entre dispositivos.

## 4. Limitação de Responsabilidade (Disclaimer)
- **Uso Offline**: O uso do Saldo no modo offline traz o risco de perda de dados se o dispositivo for danificado, perdido ou desinstalado antes de realizar uma sincronização.
- **Lembretes**: Os lembretes criados são estritamente locais ao dispositivo onde foram criados e não serão transferidos para outros dispositivos.
- **Integridade dos Dados**: Embora implementemos salvaguardas, não garantimos imunidade contra erros humanos (exclusões acidentais) ou falhas críticas de rede que possam causar inconsistências temporárias na sincronização.
- O Saldo não se responsabiliza por perda de dados decorrente da falta de sincronização pelo usuário, erros do usuário (erros ao excluir) ou conflitos de sincronização decorrentes do uso concorrente em múltiplos dispositivos.

## 5. Política de Sincronização
- **Funcionamento**: No Plano Gratuito, o usuário deve pressionar manualmente o botão de sincronização. No Plano Premium, a sincronização é automática.
- **Conflitos de Dados**: Em caso de edições concorrentes, o sistema aplica uma lógica de última escrita vence (Last-Write-Wins) com base no timestamp de atualização.

## 6. Política de Backup e Recuperação
- **Backup**: O backup depende da sincronização com a nuvem (Manual para o Gratuito, Automática para o Premium).
- **Recuperação**: A restauração baixa clientes e dívidas. Preferências locais e lembretes NÃO são restaurados.

## 7. Política de Retenção de Dados
- **Dados Excluídos**: As exclusões são permanentes após a sincronização. Não há processo de recuperação para dados deliberadamente excluídos pelo usuário.

## 8. Modificações
Reservamo-nos o direito de modificar estes termos. O usuário será notificado sobre mudanças relevantes e será solicitada uma nova aceitação.
